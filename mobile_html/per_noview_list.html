배경 Cap 이미지
  • 이 서비스 기능은 기관(회사)이 자신의 이력서 내용에 대해 열람하지 못하도록 설정하는 기능입니다.
  • 기관(회사)명을 입력하신 후 [검색버튼]을 클릭하시면 됩니다.
  • 검색조건에 해당되는 기관이 있을 경우 검색된 기관(회사) 목록에 출력됩니다.
  • 이력서 열람제한을 두고싶은 해당 기관(회사)에 대해 리스트에서 체크선택 하신 후 [열람제한 설정] 버튼을 클릭하시면 열람불가 기관목록으로 옮겨지게 됩니다.

이력서열람불가 기관 검색결과 리스트

  • 위 기관명 검색에 입력한 기관(회사)명에 해당 검색된 결과 리스트입니다.
  • 열람제한을 두고싶은 기관일 경우 검색결과 리스트에서 열람제한에 원하시는 기관(회사)명을 선택체크한 후 [열람제한설정] 버튼을 클릭하시면 됩니다.
